# Pacotes de atualização

Cada arquivo `.zip` desta pasta é uma versão empacotada do aplicativo, distribuída via
atualização over-the-air.

O arquivo `version.json` na raiz do repositório indica qual é a versão atual e aponta para
o pacote correspondente. Os clientes comparam a versão instalada com a publicada aqui e
baixam o pacote apenas quando há uma versão mais recente.

A integridade de cada pacote é verificada por checksum SHA-256.
